# CloudSewa Company Profile

## 1. Company Overview

CloudSewa is a fictional hosting and cloud infrastructure provider that offers web hosting, VPS hosting, domain registration, DNS management, SSL certificate setup, email hosting, website migration, backup support, and technical support for individuals, startups, agencies, and small businesses.

CloudSewa focuses on affordable hosting, local support, easy onboarding, transparent billing, and reliable infrastructure for Nepali customers who need websites, web applications, email services, and cloud servers.

## 2. Core Services

CloudSewa provides the following services:

1. Shared web hosting
2. WordPress hosting
3. VPS hosting
4. Managed VPS hosting
5. Domain registration and renewal
6. DNS management
7. SSL certificate installation
8. Business email hosting
9. Website migration
10. Backup and restore support
11. Basic server monitoring
12. Hosting support tickets
13. Malware cleanup assistance for managed plans
14. cPanel hosting
15. Custom cloud server setup for small businesses

## 6. Support Priority Philosophy

CloudSewa prioritizes support based on business impact.

Highest priority cases:

1. Server down
2. Website completely unreachable
3. Email service completely down
4. SSL expired on production website
5. DNS misconfiguration causing outage
6. Billing suspension affecting active service
7. Security compromise or malware infection
8. Data restore request after accidental deletion

Lower priority cases:

1. General feature questions
2. Plan upgrade inquiry
3. New domain suggestion
4. Non-urgent migration request
5. UI guidance
6. Custom configuration learning request

## 7. Important Support Principle

CloudSewa support must always identify:

1. Which service is affected
2. Whether the issue is on CloudSewa infrastructure, customer configuration, third-party DNS, registrar, code, plugin, payment, or external network
3. Whether the issue is urgent
4. Whether the customer has managed or unmanaged service
5. Whether support can fix the issue directly or only guide the customer
6. Whether the case requires escalation

Support agents must not promise server-level changes for unmanaged VPS unless the customer has purchased managed support.
