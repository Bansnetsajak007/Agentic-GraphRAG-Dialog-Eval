# CloudSewa Romanized Nepali Support Tone Guide

## 1. Purpose

This guide defines how CloudSewa support should answer customers in Romanized Nepali, English, and mixed technical language.

Hosting customers often use mixed terms such as DNS, SSL, VPS, cPanel, domain, invoice, server down, and backup. Support should preserve technical clarity while keeping the answer understandable.

## 2. Tone Principles

Support replies should be professional, calm, technical but understandable, clear about responsibility, policy-grounded, not overpromising, not blaming customer, and helpful with next steps.

## 3. Preferred Style

Use simple Romanized Nepali with technical terms in English.

Good terms:

1. domain
2. DNS record
3. nameserver
4. SSL
5. VPS
6. server
7. invoice
8. backup
9. restore
10. migration
11. propagation
12. managed support
13. unmanaged VPS
14. service ID
15. ticket

## 4. Clarification Rule

If important details are missing, ask for details.

Customer:
"site chalena"

Response:
"Kripaya domain name or service ID share garnu hola. Ma DNS, hosting status, SSL, invoice, ra server response check garna help garchu."

Customer:
"SSL problem cha"

Response:
"Kripaya domain name ra browser ma dekhiyeko SSL error screenshot share garnu hola. DNS recently change gareko cha ki chaina pani confirm garnu hola."

Customer:
"payment gare service active vayena"

Response:
"Kripaya invoice number, payment screenshot, transaction ID, amount, ra payment method share garnu hola. Billing team le verify garera activate garnecha."

## 5. Do Not Overpromise

Do not say:

1. "Server 5 minute ma fix huncha" unless confirmed.
2. "Data 100% recover huncha" unless backup verified.
3. "Domain sure recover huncha" unless registry confirms.
4. "DNS instant update huncha."
5. "Unmanaged VPS ma sab setup gardinchum."
6. "Refund instant huncha."

Use:

1. "verify garera update dinchu"
2. "backup available cha ki check garnu parcha"
3. "registry status anusar depend garcha"
4. "DNS propagation time lagna sakcha"
5. "managed support scope ma parcha ki check garnu parcha"
6. "refund method anusar timeline farak huncha"

## 6. Standard Support Examples

### Server Down

Customer:
"mero server down cha"

Response:
"Sorry for the impact. Kripaya VPS ID or IP share garnu hola. Ma infrastructure status, invoice status, network, ra VPS power state check garera update dinchu."

### DNS Propagation

Customer:
"DNS update gareko website open vayena"

Response:
"DNS propagation time lagna sakcha. Kripaya domain, changed record, old value, new value, ra change gareko time share garnu hola. Ma DNS status check garna help garchu."

### Domain Expired

Customer:
"domain expire bhayo recover huncha?"

Response:
"Domain recovery TLD ra current registry status anusar depend garcha. Kripaya domain name share garnu hola, ma grace, redemption, or deleted status check garna help garchu."

### Unmanaged VPS

Customer:
"VPS ma nginx install gardinu"

Response:
"Tapai ko VPS unmanaged ho bhane OS ra software setup customer responsibility huncha. Managed support add-on linu bhaye hamro team le setup ma help garna sakcha."

### Backup Restore

Customer:
"website delete bhayo restore garna parcha"

Response:
"Kripaya service ID, domain, ra kun date ko backup bata restore garna cha share garnu hola. Backup available cha ki check garera restore option confirm garchu."
